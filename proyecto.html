Proyectos y Portafolio

Conoce una muestra del talento de nuestros estudiantes a través de diversos proyectos: desde plataformas web hasta aplicaciones móviles, diseños digitales, soluciones de marketing y más.

Tienda Online Responsive

Proyecto de comercio electrónico con carrito de compras, login y gestión de productos usando PHP y MySQL.

App de Agenda Personal

Aplicación móvil desarrollada con Android Studio para gestionar tareas diarias y recordatorios importantes.

Mockup de Aplicación

Diseño de interfaz de usuario con Figma, centrado en la experiencia del usuario para una app de turismo local.

Sistema de Inventario

Backoffice completo para controlar stock, usuarios y reportes. Hecho con Laravel y MySQL.

Campaña en Redes Sociales

Estrategia de marketing digital con branding, creación de contenido y análisis de engagement para emprendimientos locales.

Sistema Web de Biblioteca

Proyecto integrador con frontend, backend, base de datos y diseño UI/UX para la gestión digital de bibliotecas académicas.

Simulador de Seguridad Web

Proyecto que analiza vulnerabilidades básicas en sitios y enseña prácticas seguras como hashing de contraseñas y validación.

Landing Page para Startup

Página de presentación para startups con animaciones, formularios de contacto y diseño moderno con Bootstrap y JS.

Sistema CRUD de Estudiantes

Sistema para registrar, actualizar y eliminar datos de estudiantes con validaciones, alertas y responsive design.
